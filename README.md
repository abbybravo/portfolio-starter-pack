# Portfolio Starter Pack 🚀

> Free, beginner-friendly portfolio templates designed for college students applying to internships.

Portfolio Starter Pack is an open-source collection of professional portfolio templates built specifically for students entering the job market.

Whether you are applying for:

- Software Engineering internships
- Product Management internships
- Mechanical Engineering roles
- Data Science positions
- Research opportunities
- Design roles

this repository provides a polished portfolio foundation that you can customize and deploy in minutes.

The goal:

**Edit one file. Deploy. Share with recruiters.**

---

## ✨ Features

- Three professionally designed portfolio templates
- Built with modern frontend technologies
- Beginner-friendly customization
- Fully responsive designs
- Accessibility-first components
- SEO optimized
- Static deployment with Vercel
- No backend required
- No database required
- No authentication required

---

# 🎨 Templates
